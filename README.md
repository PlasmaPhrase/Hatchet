# Hatchet
A content-based Balatro mod that adds new Jokers, consumables, and more!

# Dependencies
- [Steamodded](https://github.com/Steamodded/smods) (Required)
- [Talisman](https://github.com/SpectralPack/Talisman) (Required)

This mod may not be compatible with other pre-existing mods. 

# Current Features
- 47 Jokers
- 2 Decks
- 5 Boss Blinds
- 13 Consumables
- 1 custom challenge

# Planned Features
- None (on indefinite hiatus)

# Known Bugs
- None!
