# Hatchet
Vanilla-styled Balatro mod that adds new content!

# Dependencies
- [Steamodded](https://github.com/Steamodded/smods) (Required)
- [Talisman](https://github.com/SpectralPack/Talisman) (Recommended)

This mod may not be compatible with other pre-existing mods. 

# Current Features
- 40 Jokers
- 2 Decks
- 10 Consumables

# Planned Features
- 120 total Jokers
- 40 total Consumables
- 3 total Decks
- 3 Boss Blinds
- 2 custom Challenges
- 1 skip tag

# Known Bugs
- _Wheelbarrow_ does not indicate Mult gain
- Sephirot packs return with ERROR
- Some Sephirot packs only contain Jokers, for some reason?
