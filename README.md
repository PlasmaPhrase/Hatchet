# Hatchet
A content-based Balatro mod that adds new Jokers, consumables, and more!

# Dependencies
- [Steamodded](https://github.com/Steamodded/smods) (Required)
- [Talisman](https://github.com/SpectralPack/Talisman) (Required)

This mod may not be compatible with other pre-existing mods. 

# Current Features
- 40 Jokers
- 2 Decks
- 10+ Consumables

# Planned Features
- 120 total Jokers
- 40 total Consumables
- 3 total Decks
- 3 Boss Blinds
- 2 custom Challenges
- 1 skip tag

# Known Bugs
- _Wheelbarrow_ and _Diary Entry_ do not indicate Mult gain
- _Loss Leader_ does not work
- Some Sephirot packs only contain Jokers, for some reason?
